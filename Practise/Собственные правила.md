Правила в Wazuh хранятся в файле `/var/ossec/etc/rules/`

Создать свое правило можно 2-мя путями:
1) Перейти в директорию на сервере и создать файл .xml с прописанным правилом;
2) В Wazuh Dashboard перейти в `Server Management -> Rules -> Add new rules file`

При написании правила необходимо придерживаться определённого синтаксиса (ниже приведён пример простого правила по поиску слова в поле):
```
<!-- категоризация правил -->
<group name="windows,sysmon,cipher"> 

<!-- идентификатор правила, а также его severity -->
	<rule id="100002" level="13"> 
	
<!-- Фильрует только события создания процесса -->	
	<if_sid>61603</if_sid> 

<!-- правило будет срабатывать, если в поле image будет найдено вхождение cipher.exe с помощью регулярных выражений, которые используются с помощью указания pcre2  -->
	<field name="win.eventdata.image" type="pcre2">cipher.exe</field>
	
	<!-- описание правила -->
	<description>Possible cipher.exe execution detected </description>
	</rule>
</group>
```
